    ## 📌 The Problem: The $1.5 Billion Opportunity Gap
    In New York City, over **$1.5 Billion** in municipal aid goes unclaimed annually. Why? Because the path to eligibility is buried in fragmented systems and 80-page bureaucratic PDFs. 
    - **The Burden:** Students spend hours searching for help with no clear answers.
    - **The Impact:** 82% higher dropout risk for students with unmet basic needs.
    - **The Gap:** Students don't fail eligibility; they fail the paperwork.
    
    ## 🚀 The Solution: AlfaTech Civic Decision Engine
    AlfaTech is not a chatbot—it is **Civic Middleware**. We transform fragmented policy into executable, deterministic logic.
    
    ### ⚙️ Core Innovation: The Civic Ledger
    Unlike standard AI assistants that 'guess' or hallucinate rules, our system uses a **Deterministic Rule Engine**. 
    1. **AI Layer (Gemini 1.5 Flash):** Acts as the universal parser, extracting variables (GPA, Credits, Income) from messy transcripts.
    2. **Logic Layer (Python):** Hard-coded government rules that perform 100% accurate mathematical validation.
    3. **The Logic Trace:** Every decision is transparent. The system explains *exactly* why you qualified, citing the specific NYC HRA or CUNY policy used.
    
    ---
    
    ## 🛠️ How It Works (Technical Pipeline)
    1. **Ingest:** User uploads an Unofficial Transcript, FAFSA Summary, or Paystub.
    2. **AI Extraction:** Gemini 1.5 Flash parses unstructured data into a structured JSON schema.
    3. **Ledger Validation:** The **Civic Ledger** applies hard-coded city/university thresholds.
    4. **Active Agency:** The system queries **NYC Open Data APIs** to map food pantries and curate **Citywide Jobs** based on the student's major.
    
    ## 💻 Technical Stack
    - **Frontend:** [Streamlit](https://streamlit.io/) (Python-based UI)
    - **AI Engine:** [Google Gemini 1.5 Flash](https://ai.google.dev/) (Parsing & Intent)
    - **Decision Engine:** Custom Python Civic Ledger (Rule-based processing)
    - **Data Layer:** [Socrata API](https://dev.socrata.com/) (NYC Open Data Integration)
    - **Language:** Python 3.10+
